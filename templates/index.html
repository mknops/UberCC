Load the login data from a file

This API loads the login data from a file (provided as a GET parameter on the url), parses it, and returns the forecast for May 1 2012 - May 15 2012. The forecasts are displayed as a series of table-based plots for each hour of each day in the interval.

Process streamed login data.

This API uses the same forecasting engine as the load from file API but with streamed data. A fake stream is set up (just a loop through the provided data) and forecasts are delivered as they become available. Because the basic forecast is for any week given the input data, all that is displayed is the counts per hour for one week.
